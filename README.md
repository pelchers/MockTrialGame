This repository starts as a planning and research workspace for `MockTrial.app`, a browser-based multiplayer mock trial simulation platform.

Current focus:
- Foundational legal-procedure research
- Product definition and scope control
- Architecture option comparison before stack lock-in
- AI role simulation strategy
- Data, session, and safety blueprinting
